## Como compilar
### Dependências
- [W64-DevKit](https://github.com/skeeto/w64devkit) - Ambiente de desenvolvimento para C/C++.
- [RayLib](https://github.com/raysan5/raylib) - Biblioteca gráfica para desenvolvimento de jogos.
- [RayGui](https://github.com/raysan5/raygui) - Extensão da RayLib para desenvolver interfaces.

### Compilação
> [!TIP]
> Não é obrigatório possuir git em sua máquina. Pode-se baixar o .ZIP diretamente deste repositório e prosseguir a partir da segunda linha.
#### Windows e Linux
```bash
git clone https://github.com/baraunab/logicus-atp/
cd logicus-atp
bash ./configurar.sh
make
./build/logicus.exe
```
